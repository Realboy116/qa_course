Первая часть первого  ДЗ
Linux terminal (GitBash) commands

1) Посмотреть где я : pwd
2) Создать папку    : mkdir new_folder
3) Зайти в папку    : cd new_folder
4) Создать 3 папки  : mkdir {dir1,dir2,dir3}
5) Зайти в любоую папку : cd dir1
6) Создать 5 файлов (3 txt, 2 json) : touch file_txt_1.txt : touch file_txt_2.txt : touch file_txt_3.txt : touch file_json_1.json : touch file_json_2.json 
7) Создать 3 папки  : mkdir {folder_1,folder_2,folder_3}
8. Вывести список содержимого папки : ls (содержимое каталога) : ls -l (содержимое каталога с более подробной информацией) : ls -a (включает показ скрытых файлов) : ls -la
9) + Открыть любой txt файл   : vim file_txt_1.txt
10) + написать туда что-нибудь, любой текст. : (файл уже открыт командой "vim file_txt_1.txt") раскладка клавиатуры англ. нажимаем клавишу i (печатаем текст)
11) + сохранить и выйти. : клавиша esc :wq клавиша enter (для windows)
12) Выйти из папки на уровень выше : cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. : mv file_txt_1.txt folder_1 : mv file_json_1.json folder_1
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. : cp file_json_1.json file_txt_1.txt /C/qa/terminal/new/folder/dir1/
15) Найти файл по имени : find . -name "file_json_1.json
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. : tail -f file_txt_1.txt  (выйти Ctrl + C для ОС Windows)
17) вывести несколько первых строк из текстового файла : head -3 file_txt_1.txt
18) вывести несколько последних строк из текстового файла : tail -2 file_txt_1.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. : less file_txt_1.txt
20) вывести дату и время : date
